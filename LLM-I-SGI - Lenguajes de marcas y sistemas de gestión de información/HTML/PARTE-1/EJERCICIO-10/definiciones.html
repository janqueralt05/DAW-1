Términos Clave de Informática

HTML
Lenguaje de marcado para la creación de páginas web.
CSS
Lenguaje utilizado para describir la presentación de un documento HTML.
JavaScript
Lenguaje de programación que permite crear contenido interactivo en una página web.
Algoritmo
Conjunto de instrucciones o pasos a seguir para resolver un problema o realizar una tarea.
Base de datos
Conjunto estructurado de datos que permite el almacenamiento y la recuperación de información de manera eficiente.
